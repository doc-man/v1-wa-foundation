Sample content for tooltip:

<button type="button" class="btn btn-default help-tip"> Icon <span class="glyphicon glyphicon-question-sign"></span> <span class=" help-tip-content ">This is the inline help tip! <a onclick="event.stopPropagation()" target="_blank" href="https://www.google.co.in">Google</a></span> </button>

Class help-tip should be in parent element and it is a responsibility of programmer. Hovering on this element, child element having class help-tip-content will be shown as a tooltip.

To edit tooltip content you need to put html string as shown in highlighted pert of above mentioned sample content.
